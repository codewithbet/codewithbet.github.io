# 🚀 Betül's Personal Portfolio & Learning Diary

Kişisel web geliştirme yolculuğumu, projelerimi ve notlarımı paylaştığım statik portfolyo web sitem. Tamamen baştan sona HTML, CSS ve vanilla JavaScript kullanılarak sıfırdan geliştirildi.

## ✨ Özellikler

*   **🌓 Dinamik Gece/Gündüz Modu:** Kullanıcı tercihini tarayıcı hafızasında (`localStorage`) tutan, baştan aşağı özel tasarlanmış karanlık tema.
*   **📱 Tam Duyarlı (Responsive):** Mobil, tablet ve masaüstü ekranlara kusursuz uyum sağlayan akıcı tasarım.
*   **📖 Öğrenme Günlüğü:** Yazılım serüvenimde öğrendiklerimi ve gelişimimi not aldığım özel log bölümü.
*   **💻 Proje Vitrini:** Üzerinde çalıştığım projelerin detaylı bir şekilde sergilendiği alan.

## 🛠️ Kullanılan Teknolojiler

*   **HTML5** (Semantik ve erişilebilir yapı)
*   **CSS3** (Özel UI tasarımları, Flexbox/Grid yapısı, yumuşak geçiş animasyonları)
*   **JavaScript (ES6+)** (Tema yönetimi ve DOM manipülasyonu)
